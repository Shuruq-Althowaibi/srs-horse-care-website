# SRS Horse Care Website

SRS Horse Care is a static website for horse care services. The project includes pages for Home, About, Services, Catalog, Appointment, Privacy Policy, and Terms of Service.

## Project Files

- `index.html` - Home page
- `About.html` - About page
- `Services.html` - Services page
- `Catalog.html` - Image catalog / gallery page
- `Appointment.html` - Appointment request page
- `PrivacyPolicy.html` - Privacy policy page
- `TermsOfService.html` - Terms of service page
- `style.css`, `style5.css`, `stylePT.css`, `stylesheet2.css` - Styling files
- `Script.js`, `script1.js` - Website interaction scripts
- `logo.png` and image files - Website images and catalog assets

## How To Run

1. Open the `website` folder.
2. Double-click `index.html`.
3. The website should open in your browser.

You can also open any page directly, such as `Catalog.html` or `Appointment.html`.
